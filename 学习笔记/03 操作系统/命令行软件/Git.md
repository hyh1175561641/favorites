# 仓库管理

```
git clone #克隆仓库
git init #初始化
git add #添加文件
git add . #添加全部文件
git status #提交之后是否有修改
git diff #详细信息
git commit #将缓存区的内容提交到仓库中
git reset #取消已缓存的内容
git rm #从git中移除某个文件
git mv #移动或重命名某个文件
git tag # 打标签
```

# 分支管理

HEAD相当于一个指针，指向当前版本

```
git branch dev 新建dev分支
git branch -d dev 删除dev分支
git checkout dev 切换分支
git checkout master 切换回主分支
git merge dev 把dev分支合并到了主分支上
```
