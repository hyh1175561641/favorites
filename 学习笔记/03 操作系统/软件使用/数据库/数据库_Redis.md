# Redis概述

Redis是一种NoSQL技术，它的性能十分优越，可以支持每秒几十万次的读写操作，性能远超数据库，并且还支持集群、分布式、主从同步等配置，原则上可以无限扩展，让更多的数据存储在内存中，更让人欣慰的是它还支持一定的事务能力，这保证了高并发的场景下数据的安全和一致性

**NoSQL技术**

这是一种基于内存的数据库，且提供一定的持久化功能。

Redis和MongoDB是当前使用最广泛的NoSQL
